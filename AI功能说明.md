# AI智能功能说明文档

### 1. 噪音类型智能识别

功能描述：
- 实时识别环境中的噪音类型
- 支持识别：说话声、桌椅移动、手机铃声、脚步声、键盘声、环境噪音等
- 显示识别置信度

技术实现：
 使用音频特征提取（RMS、过零率、频谱质心、MFCC等）
 基于规则的分类算法（可扩展为机器学习模型）
 实时分析音频数据流

使用方法：
1. 开始监控后，系统自动识别噪音类型
2. 在状态面板的"噪音类型识别"卡片中查看实时结果
3. 显示格式：图标 + 类型名称 + 置信度

2. 智能阈值推荐

功能描述：
- 基于历史数据分析，推荐最佳噪音阈值
- 考虑班级类型、时间段等因素
- 提供置信度和分析说明

技术实现：
- 统计分析历史录制数据
- 分析阈值与警告次数的关系
- 找到警告次数适中的最佳阈值（1-2次警告为最佳）

使用方法：
1. 点击"智能阈值推荐"按钮
2. 系统分析历史数据并显示推荐值
3. 点击"应用推荐值"按钮应用新阈值

推荐逻辑：
- 分析不同阈值下的平均警告次数
- 选择警告次数在1-2次之间、警告率在30-60%的阈值
- 如果数据量充足，置信度可达90%

3. 学习效率分析

功能描述：
- 分析不同时间段的噪音水平
- 识别最佳学习时段
- 分析噪音趋势（上升/下降/稳定）
- 提供个性化建议

**技术实现：**
- 时间段划分：上午（8-12点）、下午（12-18点）、晚上（18-22点）
- 计算各时间段的平均警告次数
- 对比最近数据与历史数据，分析趋势

**使用方法：**
1. 点击"学习效率分析"按钮
2. 查看时间段分析报告
3. 查看趋势分析和建议

**分析内容：**
- 各时间段的平均警告次数
- 噪音水平趋势（上升/下降/稳定）
- 最佳学习时段推荐
- 个性化改进建议

### 4. 班级纪律报告生成

**功能描述：**
- 生成完整的班级纪律统计报告
- 对比不同班级的纪律水平
- 提供数据可视化

**技术实现：**
- 统计所有录制记录
- 按班级分组分析
- 计算平均警告次数、最高警告次数等指标
- 评估纪律水平（优秀/良好/一般/需改进）

**使用方法：**
1. 点击"生成纪律报告"按钮
2. 查看完整的统计报告
3. 对比各班级的纪律水平

**报告内容：**
- 总录制次数
- 平均警告次数
- 各班级详细数据
- 纪律水平评估


##  扩展建议

- 可以训练深度学习模型提高识别准确率
- 使用TensorFlow.js加载预训练模型
- 支持在线学习和模型更新

### 2. 更多噪音类型
- 扩展识别更多噪音类型
- 支持自定义噪音类型
- 学习用户标注的数据

### 3. 高级分析
- 噪音模式预测
- 行为异常检测
- 学习效果关联分析

## 使用注意事项

1. **性能优化**：噪音类型识别每500ms更新一次，避免过度计算
2. **数据要求**：智能阈值推荐需要至少10条历史数据才能提供可靠建议
3. **准确性**：当前使用基于规则的分类，准确率约60-80%，可升级为ML模型提高准确率
4. **浏览器兼容性**：需要支持Web Audio API的现代浏览器
